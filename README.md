## Description

Airbnb clone coding backend repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# development - watch mode
$ npm run start:dev

# production mode (DONT USE - db is not set yet)
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
